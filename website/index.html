What Ternix does
SSH / Telnet / Serial
Full protocol support with key-based auth and agent forwarding
Remote Desktop (RDP / VNC)
Integrated RDP and VNC support for seamless graphical sessions
SFTP Dual-Pane Manager
Drag-and-drop file transfers between local and remote
Session Recording
Record and replay any terminal session, exportable
Port Forwarding
Local, remote, and dynamic tunnels with one click
Split Panes
Horizontal and vertical splits, resize any pane
Tab Management
Named tabs, color-coded groups, keyboard shortcuts
Connection Profiles
Saved sessions with per-profile keys, colors, and env vars
Cross-Platform
Runs natively on Windows and Linux
