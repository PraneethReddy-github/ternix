Quick Start

Up and running in minutes

PowerShell / Terminal 
# Download the installer and run it
Start-Process "https://github.com/PraneethReddy-github/ternix/releases/latest/download/Ternix-Setup-0.1.1.exe"

# Or use winget (coming soon)
winget install Ternix
  1. Download Ternix-Setup.exe (NSIS installer)
  2. Run the installer and follow the on-screen instructions
  3. Launch Ternix from Start Menu or Desktop
Bash 
# Download the .deb package
wget "https://github.com/PraneethReddy-github/ternix/releases/latest/download/ternix_0.1.1_amd64.deb"

# Install with apt
sudo apt install ./ternix_*_amd64.deb
  1. Download the .deb package
  2. Install via sudo apt install ./ternix_*_amd64.deb
  3. Launch Ternix from your application menu
Bash 
# Download AppImage
wget "https://github.com/PraneethReddy-github/ternix/releases/latest/download/Ternix-0.1.1.AppImage"

# Make executable and run
chmod +x Ternix-*.AppImage
./Ternix-*.AppImage --no-sandbox
  1. Download the .AppImage
  2. Make it executable: chmod +x Ternix-*.AppImage
  3. Run with: ./Ternix-*.AppImage --no-sandbox
