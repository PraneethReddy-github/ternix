What Ternix does
SSH
Password, key, agent, and keyboard-interactive auth. Multi-hop jump hosts and host-key pinning
Telnet & Serial
Telnet option negotiation, and serial with full baud, parity, and flow control
Local Shells
A native PTY — your real shell, with real signals
Remote Desktop (RDP / VNC)
Graphical sessions rendered right inside a pane, with native-client fallback
SFTP Dual-Pane Manager
Drag between panes, or drop files straight from your desktop to upload
Smart Transfers
Live speed and ETA. Pause, resume, or cancel one file without stopping the rest
SSH Key Vault
Generate, import, and deploy keys — ssh-copy-id without leaving the app
Encrypted at Rest
Credentials sealed with AES-256-GCM. Optional master password. Nothing in the cloud
Port Forwarding
Local, remote, and dynamic SOCKS5 tunnels with live byte counters
Session Recording
Record to asciinema .cast and replay with a scrubbable timeline
Live System Monitor
CPU, memory, disk, network, and temperatures — for this machine or the server
Split Panes & Broadcast
Up to six panes per tab. Type once, send to every one of them
Command Palette
Ctrl+P to jump to any saved session or action
Command Snippets
Reusable commands with ${VAR} prompts, global or scoped to one session
Bring Your Sessions
Import from OpenSSH config, PuTTY, WinSCP, MobaXterm, Tabby, or CSV
Themes
12 built-in themes, a full ANSI theme builder, and custom CSS
Cross-Platform
Runs natively on Windows and Linux
